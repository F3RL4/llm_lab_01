# LLM Model Specifications

## Model Details
- **Name**: Qwen3.5-9B
- **Format**: GGUF (Quantized)
- **Developer**: Alibaba Cloud / Qwen Team
- **Architecture**: Transformer-based LLM

## Inference Setup
- **Runtime**: LM Studio

- **Hardware Target**: RTX 4050 Laptop (6GB VRAM)
